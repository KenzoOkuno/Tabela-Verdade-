# Tabela Verdade Interativa (3 Variáveis)  

Este projeto é um programa em Python que cria uma **tabela verdade interativa** para três variáveis lógicas: A, B e C.  

O programa permite que o usuário:  
- Marque ou desmarque valores de A, B e C usando caixas de seleção (checkboxes).  
- Veja imediatamente o resultado de diversas operações lógicas clássicas.  
- Trabalhe com operações como: Negação (¬), Conjunção (∧), Disjunção (∨), Disjunção Exclusiva (⊕), NAND (↑), NOR (↓), e Implicação (⇒).  

## Funcionalidades

- **Interatividade completa**: Alterações nos valores das variáveis são refletidas instantaneamente na tabela.  
- **Visualização clara**: Cada operação lógica é exibida com seu símbolo matemático correto.  
- **Todas as combinações de 3 variáveis** são mostradas.  

## Como usar

1. Certifique-se de ter o Python 3 instalado.  
2. Salve o arquivo `tabela_verdade.py`.  
3. Execute o programa no terminal ou IDE:  
   ```bash
   python tabela_verdade.py
